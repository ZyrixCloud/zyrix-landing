# Zyrix Landing (i18n EN/TR/AR) – Next.js + Tailwind – Ready for Vercel
## Included
- Localized pages: `/` (EN), `/tr`, `/ar` (RTL).
- Case Studies / Logos section (replace SVGs in `public/logos`).
- Waitlist form → `/api/subscribe` with Mailchimp/HubSpot support via env vars.
## Quickstart
1) `npm i`
2) Copy `.env.example` → `.env.local` and set keys.
3) `npm run dev` → http://localhost:3000
4) Push to GitHub, import in Vercel, set env vars.
